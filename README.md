📝 ToDo Web UI (Frontend)

A responsive and user-friendly ToDo web interface built using HTML, CSS, JavaScript, and Bootstrap.  
This frontend consumes a secure Spring Boot REST API with JWT-based authentication.

---
 
 🚀 Features

- 🎨 Clean and responsive UI (Bootstrap)
- 🔐 JWT-based authentication integration
- 📌 Create, update, and delete tasks
- ⚡ Real-time UI updates using JavaScript
- 🔗 Seamless integration with backend REST API

---

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap

---

🔗 Backend API

This UI connects to the backend:
👉 https://github.com/p-a-n-k-a-j/todo-app

---

▶️ Run Locally

1. Clone the repository
```bash
git clone https://github.com/p-a-n-k-a-j/todo-ui.git
cd todo-ui
```
2. Open in browser
```bash
open index.html
```
⚙️ Configuration
Update API base URL inside JavaScript file:
```bash
const BASE_URL = "http://localhost:8080";
```
